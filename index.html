🔗 Cloud Setup — Supabase (Free, 5 min)
1. Go to supabase.com → click Start your project → sign up free
2. Click New project → give it any name → set a password → click Create project (wait ~1 min)
3. In left menu click Table Editor → New table → name it cong_data → turn OFF “Enable Row Level Security” → click Save
    OR paste the SQL below in SQL Editor and click Run
create table if not exists cong_data (
  id integer primary key,
  payload jsonb,
  updated_at timestamptz default now()
);
4. In the left sidebar, click the Settings icon (⚙️ at the very bottom of the sidebar)
5. Click Data API (or just API depending on your version)
6. You will see two things to copy:
   • Project URL — looks like https://abcdefgh.supabase.co
   • Publishable key → tap the copy icon next to sb_publis… under “default”
7. Paste both into the fields below and tap Save & Connect
