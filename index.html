🔗 Cloud Setup — Firebase (Free, 3 min)
1. Go to console.firebase.google.com → sign in → Create a project
2. Left menu: Build → Realtime Database → Create Database → Next → Enable
3. Click Rules tab → replace all text with:
{"rules":{".read":true,".write":true}}
Click Publish
4. Go back to Data tab → copy the URL (e.g. https://project-rtdb.firebaseio.com)
5. Paste below → tap Save & Connect. Share same URL with publishers.
